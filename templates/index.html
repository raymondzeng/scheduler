Select a location to add as a dependancy...

Click on a location's text to select it.

When any locations are selected, click the green '+' next to a location to add the selected ones as dependencies.

When a location has dependencies, it means those dependencies must be visited first, but in no particular order.


Double-click a time to edit it.

The three columns are earliest, duration, and latest, respectively.

Earliest and latest are in hhmm format where hh is the hour from 0 - 24 and mm is the minutes between 0 - 60.

Duration is in minutes.

Times can be read as: I want to be here no longer than earliest for duration but no later than latest.
